Problem Description
Given an array A containing N integers.

You need to sort the array and return it.

Problem Constraints
1 <= N <= 5000

1 <= A[i] <= 103

Input Format
First and only argument is an integer array A.

Output Format
Return the sorted version of A.

Example Input
Input 1:

A = [2, 1, 3]
Input 2:

A = [4, 2, 1, 1]

Example Output
Output 1:

[1, 2, 3]
Output 2:

[1, 1, 2, 4]
